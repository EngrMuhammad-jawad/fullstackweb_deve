General Questions

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the structure and content using elements like headings, paragraphs, and links.

HTML, CSS, and JavaScript differences?

HTML builds structure, CSS handles style, and JavaScript adds interactivity. Like a house: HTML is walls, CSS is paint, JavaScript makes doors open.

Glossary of Web Terms

HTML
HyperText Markup Language — the structure of web pages
CSS
Cascading Style Sheets — the styling of web pages
JavaScript
The programming language that adds interactivity to web pages
